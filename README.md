# 🪄 WizardVision

WizardVision is an interactive Human Computer Interaction (HCI) project that uses Computer Vision and AI-based hand tracking to simulate real-time spell casting through gestures.

Built using Python, OpenCV, MediaPipe, and Pygame, the system detects hand movements from a webcam feed and triggers dynamic visual and audio spell effects such as fire, lightning, and shield spells.

## ✨ Features

* Real-time hand tracking and gesture recognition
* Fire, lightning, and shield spell effects
* Interactive sound and visual feedback
* Motion trail and wand glow effects
* Webcam-based gesture interaction

## 🛠️ Technologies Used

* Python
* OpenCV
* MediaPipe
* Pygame
* Computer Vision
* Human Computer Interaction (HCI)
